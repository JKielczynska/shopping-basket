# shopping-basket

Market checkout component with REST API.

## Features
* open new empty basket
* add product to basket
* update product in basket
* close basket and return total price
* discount mechanism

## Build

```
./gradlew clean build

```

## Technologies
* Java 8
* SpringBoot 1.5.9 RELEASE
* Gradle
* Hibernate
* Lombok
* JUnit
* Mockito
